# Changelog

All notable changes to this project will be documented in this file. See [standard-version](https://github.com/conventional-changelog/standard-version) for commit guidelines.

## [0.1.0](https://github.com/TRETYAKweb/tips-and-tricks-react/compare/v0.0.2...v0.1.0) (2023-06-15)


### ⚠ BREAKING CHANGES

* **pages:** this change may require updating some of the types and interfaces that depend on the LoginPage component.
* This change may affect code that relies on the previous implementation of the 'authReducer' function. The type of the 'action' parameter has been changed to 'AnyAction' from 'unknown'.
* **settings:** The payload of the setSettingsOpen action is now a boolean type.
* **hooks:** This commit removes the old useTipById hook implemented in JavaScript and replaces it with a TypeScript version. The id parameter is now required and must be a string.
* **hooks:** useErrorMessage now requires the useAppDispatch hook from redux-toolkit instead of useDispatch from react-redux.
* The ISignUpFormShape interface now includes confirmPassword. The ISignUp interface has been added and omits confirmPassword.
* **forms:** The type of the PublishTipForm form shape has changed to IPublishFormShape.
* The file extension of the entry point is changed to .tsx.
